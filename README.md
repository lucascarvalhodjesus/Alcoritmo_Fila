# Algoritmo Fila (Queue) em Java

Projeto desenvolvido em Java para estudo e implementação da estrutura de dados **Fila (Queue)** utilizando nós encadeados.

## Sobre o Projeto

Uma fila é uma estrutura de dados que segue o princípio **FIFO (First In, First Out)**, ou seja, o primeiro elemento inserido é o primeiro a ser removido.

Este projeto demonstra a implementação de uma fila utilizando Programação Orientada a Objetos (POO), permitindo compreender na prática o funcionamento dessa estrutura de dados fundamental.

## Tecnologias Utilizadas

- Java
- Programação Orientada a Objetos (POO)
- IntelliJ IDEA
- Git
- GitHub

## Conceitos Aplicados

- Estruturas de Dados
- Filas (Queue)
- FIFO (First In, First Out)
- Nós Encadeados
- Classes e Objetos
- Encapsulamento
- Manipulação de Referências

## Funcionalidades

- Inserir elementos na fila
- Remover elementos da fila
- Consultar o primeiro elemento
- Verificar se a fila está vazia
- Exibir os elementos armazenados
